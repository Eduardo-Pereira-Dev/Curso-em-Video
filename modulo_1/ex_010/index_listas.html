Trabalhando com listas

Listas Ordenadas:

Utilizando a tag <ol> podemos criar uma lista ordenada e dentro do elemento 'ol' temos as tags <li> para cada item da lista.

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes
  5. Ir para a faculdade

Listas Não-Ordenadas:

Utilizando a tag <ul> podemos criar uma lista não-ordenada e dentro do elemento 'ul' temos as tags <li> para cada item da lista.

  • Pão
  • Leite
  • Manteiga
  • Arroz
  • Feijão
  • Mistura

Juntando Listas:

Podemos colocar uma lista dentro de outra lista, no exemplo a baixo temos uma lista ordenada e dentro dela temos uma lista não-ordenada.

  1. Linguagem Antiga:
    • Fortran
    • Visual Basic
    • Delphi
  2. Linguagem Nova:
    • Java Script
    • Java
    • Python

Lista de Definições:

Utilizando a tag <dl> podemos criar uma lista de definição. Dentro do elemento 'dl' teremos outras duas tags, <dt> que indica o termo da lista e <dd> que indica a descrição do termo.

Ex:Termo
Ex:Descrição

Linguagem de marcação para criação de conteúdo de um site
