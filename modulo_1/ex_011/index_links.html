Utilizando Links

Link Externos:

Utilizando a tag <a href="URL"> podemos adicionar links no nosso site, porém ele não irá abrir o link em uma nova página. Essa forma de link é mais utilizando para navegar pelas páginas de um mesmo site.

Você pode acessar o meu Instagram clicando aqui

Utilizando a tag <a href="URL" target="_blank" rel="external"> podemos adicionar links no nosso site, porém ele irá abrir o link em uma nova página. Essa forma de link é mais usada para navegar pelas páginas de sites diferentes.
